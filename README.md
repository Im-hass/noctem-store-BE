# Cafe Noctem Project

## 🍜 프로젝트 설명
1. 카페 모바일 애플리케이션 주제의 프로젝트 입니다.
2. Spring Boot 기반의 백엔드 서버입니다.
3. 프로젝트 기간 : 2022.09.13 ~ 진행중 
4. Java 11 
5. Spring Boot 2.6.8
6. Gradle 7.5.1
